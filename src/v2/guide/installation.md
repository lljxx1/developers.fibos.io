FIBOS 支持常用的 UNIX 操作系统，比如 Mac OSX, Linux 和 FreeBSD。
FIBOS 集成了FIBJS的 JavaScript CLI 命令行控制台功能。


## 快速安装


``` bash
# 最新稳定版
curl -s https://fibos.io/download/installer.sh |sh
fibos --version
```
